用Gridea和Github创建博客

  1. 2020-08-26
  2. 264字
  3. 2 min read

材料

Gridea官网
Github账号
创建如下：
先要有个邮箱（如QQ邮箱）
然后打开Github官网

点击sign up

输入用户名Username
邮箱Email address
密码Password

然后点击验证
按提示操作即可

最后点击Create account即可

然后依次选择
Student
A little
Learn to code
Host a project(repository)
Create a website with Github Pages
点击Complete setup
然后打开邮箱，点击Github发过来的邮箱

即可完成注册。

创建仓库

打开Github官网

创建博客

打开下载好的Gridea
点击远程在域名中输出

自己Github用户名.github.io

仓库名称同上
分支填master
仓库用户名填自己Github用户名
邮箱填注册Github账号的邮箱
令牌填Github自己账号的密码
点击保存

主题更换


如果觉得Gridea自带的主题不好看（就像我），可以点击更多主题。

同步

更改Gridea后，点击预览查看效果
点击同步或同步下面的世界图标

成果

我搭建的博客
